# AudioVisualizer
✅ 极低资源占用 - Direct2D硬件加速，CPU占用<1%

✅ 实时响应 - 10ms刷新间隔，无延迟感

✅ 清晰可视 - 左声道绿色，右声道蓝色，易于辨识

✅ 音量归一化 - 使用RMS和分贝转换，准确反映音量

✅ 置顶显示 - 窗口默认置顶，游戏时也能看到



| 操作                      | 行为           |
| ------------------------- | -------------- |
| **无 Ctrl**               | 完全穿透 ✅     |
| **Ctrl + 中间拖动**       | 移动窗口 ✅     |
| **Ctrl + 靠近左边缘拖动** | 向左缩放宽度 ✅ |
| **Ctrl + 右下角拖动**     | 同时调整宽高 ✅ |
| **Ctrl+Alt+V**            | 显示/隐藏 ✅    |
