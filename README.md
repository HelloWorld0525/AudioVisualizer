# AudioVisualizer

| 操作                      | 行为           |
| ------------------------- | -------------- |
| **无 Ctrl**               | 完全穿透 ✅     |
| **Ctrl + 中间拖动**       | 移动窗口 ✅     |
| **Ctrl + 靠近左边缘拖动** | 向左缩放宽度 ✅ |
| **Ctrl + 右下角拖动**     | 同时调整宽高 ✅ |
| **Ctrl+Alt+V**            | 显示/隐藏 ✅    |
